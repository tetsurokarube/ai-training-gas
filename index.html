1. 導入：AIとGASで業務はどう変わる？

「Gemini」とは？

Googleが開発した最新の生成AI（人工知能）です。自然な文章を理解し、要約、翻訳、そしてプログラミングコードの生成を行います。「こんな風に動くプログラムを作って」とお願いするだけで、正確なコードを書き上げてくれる「専属エンジニア」のような存在です。

「GAS」とは？

Google Apps Scriptの略で、Googleが提供しているプログラミングを実行するための無料の仕組みです。最大の強みは、Gmail、スプレッドシート、GoogleカレンダーなどのGoogleサービス同士を簡単に連携して、手作業を自動化できる点です。

1. 導入：なぜ今、この組み合わせなのか？

日々の業務で「これ、いちいちコピペするの面倒だな…」と思ったことはありませんか？

  • カレンダーの予定を見て、朝礼用のシートに手入力で書き写す
  • スプレッドシートのリストから、顧客あてのメール下書きを一括作成する

今までは、自動化するには「プログラミングをイチから学習する」高い壁がありました。しかし今は、「Geminiに日本語でお願いする」だけで、誰でもGASのツールを作れるようになりました。今日はこの組み合わせで自動化を体験しましょう。

2. 座学：Geminiへの指示（プロンプト）のコツ

AIは「曖昧な指示」が苦手です。
プログラミングのコードを生成してもらう際には、以下の3つの要素を含めると成功確率が格段に上がります。

  • 👤 役割：「あなたはプロのGoogle Apps Scriptのエンジニアです」
  • 🎯 目的：「自分のGoogleカレンダーの今日の予定を取得して」
  • 📝 条件：「コードには日本語で丁寧なコメント（説明）を入れて」

3. ワーク1：Geminiにコードを書かせる（1/2）

ミッション：「空っぽのスプレッドシートに、自分の本日の予定（Googleカレンダー）を自動で一覧化する」

以下の文をコピーして、Geminiに送ってください
あなたはプロのGoogle Apps Script（GAS）エンジニアです。
以下の要件を満たすGASのコードを作成してください。

【要件】
・自分のGoogleカレンダーから、本日の予定（イベント）をすべて取得してください。
・現在開いているスプレッドシートのアクティブなシートに書き出してください。
・1行目に「見出し（開始時刻、終了時刻、予定のタイトル）」を太字で入力してください。
・2行目から、時系列順に予定を書き出してください。時刻は「HH:mm」形式にしてください。
・関数の名前は「getTodaysEvents」にしてください。
・初学者でも何をしているか分かるように、コードの各行に丁寧な日本語のコメントをつけてください。

3. ワーク1：Geminiにコードを書かせる（2/2）

💡

Geminiの回答が出たら

Geminiがプログラムの文字列（コードブロック）を出力します。
そのコードブロックの右上にある「コードをコピーする」ボタン（またはコピーのアイコン）を押して、クリップボードに保存しておいてください。

4. ワーク2：実際に動かしてみる（ファイル準備）

いよいよ、AIが作成したプログラムをご自身のスプレッドシートに組み込みます。

Step 1. 新しいスプレッドシートとエディタの準備
  1. ブラウザで「Google ドライブ（またはスプレッドシート）」を開き、**「新規」スプレッドシート**を作成します。（真っ白なシートで構いません）
  2. 上部のメニューバーの [拡張機能] ＞ [Apps Script] をクリックします。
  3. 「無題のプロジェクト」というエディタ画面が開いたら、最初から書かれている function myFunction() { } の文字を全て削除します。
  4. 先ほどGeminiからコピーしたコードを貼り付けます。
  5. 画面上部の「💾 保存アイコン」をクリックして保存します。
8

4. ワーク2：実行とセキュリティ承認

初めて実行するときだけ、Googleから「スプレッドシートやカレンダーを操作してもいいですか？」という警告が出ます。以下の手順で許可してください。

Step 2. セキュリティの承認手順（最初だけ）
  1. 画面上部の [▷ 実行] ボタンをクリックします。
  2. 「承認が必要です」という画面が出たら [権限を確認] をクリック。
  3. 自分のGoogleアカウントを選択します。
  4. 「このアプリはGoogleで確認されていません」と出たら、左下の [詳細] を開き、[無題のプロジェクト（安全ではないページ）に移動] をクリックします。
  5. 最後に [許可] をクリックします。
9

4. ワーク2：成功確認 🎉

🎉

成功しましたか？

スプレッドシートの画面に戻ってみましょう。
何もない真っ白だったシートに、あなたの「今日のカレンダーの予定」がズラッと一覧で入力されていれば大成功です！

※もし今日の予定がない場合は、見出しだけが作られます。
テスト的に今日のカレンダーに予定を入れてから再度実行してみてください。

5. まとめ・注意点

たったこれだけの手順でプログラムを作成できました。「明日の予定を毎晩Chatworkに送る」なども同様の手順で簡単に作れます。

⚠️

業務で使う際の絶対に守るべき注意点

  1. 機密情報をGeminiに入力しない：
    実際の顧客名や売上数字などは絶対にプロンプトに入れないでください。処理のルールだけをAIに作らせて、データ自体はGAS内で守りましょう。
  2. 自己責任での実行：
    AIのコードは完璧ではありません。必ずテスト・動作確認をしてから業務使用してください。
